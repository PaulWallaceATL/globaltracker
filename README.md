# GlobalTracker

God's-eye multi-sensor situational awareness globe: conflict, thermal, air, sea, traffic, weather, population, and news — searchable situations with stakeholder lenses.

## Local development

```bash
cp .env.example .env.local
# fill in API keys
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Push to `main` and deploy on Vercel. Set the same env vars as `.env.local` (see `.env.example`). Do not commit secrets.
